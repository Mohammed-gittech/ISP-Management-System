# ISP Management System 🚀

A Multi-Tenant ISP Management RESTful API built with ASP.NET Core 8.

## 🏗 Architecture

- Clean Architecture
- Clean Separation of Concerns
- Repository Pattern
- Unit of Work
- DTOs (Manual Mapping)

## 🔐 Features

- Subscriber Management
- Subscription Lifecycle
- Revenue Reports
- Dashboard Summary Metrics
- Expiring Subscription Alerts
- Authentication & Authorization
- Audit Logging
- Unit Testing (xUnit)

## 🧪 Testing

- Service Layer Tests
- Validator Tests
- Clean Test Structure

## 🛠 Tech Stack

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- xUnit
- FluentValidation
